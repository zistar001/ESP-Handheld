# 功能使用说明 — ESP-Handheld

## 一、按键说明

### 按键布局

```
        [START]  [UP]
                    ↑
    [LEFT] ←     → [RIGHT]

         [DOWN]  [A]  [B]
          ↓
```

| 按键 | 物理位置 | GPIO | 功能 |
|------|---------|------|------|
| **UP（上）** | 左上 | GPIO16 | 上方向 |
| **DOWN（下）** | 左下 | GPIO15 | 下方向 |
| **LEFT（左）** | 中左 | GPIO17 | 左方向 |
| **RIGHT（右）** | 中右 | GPIO7 | 右方向 |
| **A** | 右上 | GPIO4 | 确认 |
| **B** | 右下 | GPIO5 | 退格/删除 |
| **START** | 中下 | GPIO6 | 菜单/开始 |

### 组合键

| 组合 | 功能 |
|------|------|
| **START + B（主界面）** | 退出当前应用/返回上一级 |
| **START + B（Retro-Go）** | 从游戏系统返回本系统 |
| **START + B（XiaoZhi）** | 从AI助手返回本系统 |
| **START + A（游戏中）** | 模拟 SELECT 键 |
| **START（按住1秒，游戏中）** | 打开游戏菜单（存档/读档/退出） |
| **RIGHT（键盘模式）** | 按住通话（WiFi音频PTT） |

## 二、开机引导

### 开机后请耐心等待

设备开机后需要完成一系列初始化（约3-5秒）：
1. 屏幕亮起 → 显示 LVGL 启动画面
2. 初始化 SD 卡、音频、传感器
3. 初始化蓝牙
4. **连接 WiFi**（如果已保存密码）
5. **加载天气数据**（如果配置了 API Key）

> **⚠️ 重要：** WiFi 和天气加载完成前（状态栏 WiFi 图标完整显示前），建议不要快速操作按键，否则可能导致重启。等待约 5-10 秒待系统稳定后再使用。

## 三、界面导航

### 主屏（Launcher）

开机稳定后进入主屏，显示：
- **顶部状态栏：** WiFi 图标、当前时间、蓝牙状态、电池电量
- **中央天气卡片：** 天气预报（需配置 API Key）
- **室内温湿度：** AHT20 传感器数据（如果硬件有此模块）

按 **START** → 进入应用菜单

### 应用菜单（Menu）

9 个应用卡片，2 列布局：

| 游戏（Retro-Go） | 小智（XiaoZhi AI） |
|:---:|:---:|
| **键盘** | **鼠标** |
| **占卜（易经）** | **小六壬** |
| **计时** | **频谱** |
| **设置** | |

- **UP/DOWN：** 上/下翻页
- **LEFT/RIGHT：** 左/右移动
- **A：** 选择并进入应用
- **B/START：** 返回主屏

### 应用内导航

- **B/START：** 退出当前应用，返回菜单（大部分应用）
- **A：** 确认操作

## 四、各应用使用说明

### 游戏（Retro-Go）

进入"游戏"卡片后，设备会**重启并切换到 Retro-Go** 游戏系统（ota_0 分区）。

**支持平台：**
- NES/FC 红白机（.nes）
- GB/GBC 掌机（.gb, .gbc）
- SMS/GG 世嘉（.sms）
- PCE/TurboGrafx-16（.pce）
- SNES/SFC 超任（.smc）

**ROM 存放路径（SD 卡）：**
```
/sdcard/roms/
├── nes/        ← 放 NES/FC 游戏 (.nes)
├── gb/         ← 放 GB/GBC 游戏 (.gb, .gbc)
├── sms/        ← 放 SMS/GG 游戏 (.sms)
├── pce/        ← 放 PCE 游戏 (.pce)
└── snes/       ← 放 SNES/SFC 游戏 (.smc)
```

> **注意：** 如果某个平台的文件夹不存在，需要先在 SD 卡上手动创建。ROM 文件名建议使用英文或拼音。

**菜单导航：**
- 在 ROM 浏览器中 UP/DOWN 选择游戏
- 按 A 进入
- **START 按住 1 秒：** 游戏菜单（存档/读档/退出）

**返回本系统：** 游戏菜单选"Exit"，或 **START+B 按住**

### 小智（XiaoZhi AI）

进入"小智"卡片，设备会**重启并切换到 XiaoZhi AI** 系统（ota_1 分区）。这是一个 AI 语音助手。

**返回本系统：** 按住 **START+B 约 150ms**

### 键盘（BLE HID 键盘）

将本设备作为**蓝牙键盘**连接到手机、平板或电脑。

**使用方法：**
1. 进入"键盘"应用
2. 在目标设备上打开蓝牙，搜索 **ESP-Handheld**
3. 配对（Just Works，无需输入密码）
4. 成功后物理按键变为键盘输入

**按键映射：**
| 物理按键 | 键盘输出 |
|---------|---------|
| UP | 上方向键 ↑ |
| DOWN | 下方向键 ↓ |
| LEFT | 左方向键 ← |
| RIGHT | 右方向键 → |
| A | Enter（回车） |
| B | Backspace（退格） |
| START | Tab |

**PTT 语音：** 按住 **RIGHT** 键可像对讲机一样说话（需先设置电脑 IP）

**退出：** **START + B**

### 鼠标（BLE HID 鼠标）

通过 IMU 陀螺仪控制鼠标指针。

- **A：** 鼠标左键
- **B：** 鼠标右键
- **UP：** 提高灵敏度
- **DOWN：** 降低灵敏度
- **退出：** **START + B**

### 占卜（易经）

基于《易经》六十四卦。进入后按 A 摇卦，显示卦辞解释。

### 小六壬（Xiao Liu Ren）

传统占卜术。输入三个数字（月、日、时），按 A 推算结果。

### 计时（倒计时）

- **UP/DOWN：** 设置时间
- **A：** 开始/暂停
- **退出：** **START + B**

### 频谱（音频频谱分析仪）

实时 FFT 频谱可视化。

- **UP/DOWN：** 选择参数
- **LEFT/RIGHT / A：** 调整参数
- **退出：** **B** 或 **START**

### 设置

| 设置项 | 功能 |
|-------|------|
| 音量 | 扬声器音量（0-100） |
| 亮度 | 屏幕亮度（0-100） |
| 配置 WiFi | 进入配置模式（热点 + 网页） |
| 天气地区 | 选择和风天气地区 |
| 主题 | 切换 6 种配色 |
| 主题颜色 | 自定义 5 个颜色槽 |
| 按键映射 | 自定义 7 个按键 |
| 待机时间 | 自动息屏延时 |
| 关于 | 系统信息 |

## 五、WiFi 配置

### 方法一：设置菜单
设置 → 配置 WiFi → 设备开启热点 **ESP-Handheld_XXXX** → 手机连接 → 浏览器打开 192.168.4.1 → 选择 WiFi 并输入密码

### 方法二：自动连接（已保存密码时）
设备会自动连接之前保存的 WiFi。

### WiFi 音频（PC 端 IP 设置）
设置 → IP 输入 → 输入电脑 IP 最后一段数字
> 例：ESP 的 IP 为 192.168.1.100，电脑为 192.168.1.50 → 输入 50

## 六、蓝牙配对

1. 进入"键盘"或"鼠标"应用
2. 目标设备搜索 **ESP-Handheld**
3. 配对（Just Works 模式）

**Windows 注意：** 如果连不上，先清除所有 "ESP-Handheld" 配对记录，重启蓝牙再试。

---

## 七、电源管理

闲置一段时间（默认 5 分钟）后：
1. 屏幕息屏
2. 关闭蓝牙、WiFi
3. 进入 Light Sleep 待机（~5-10mA）

**按任意按键唤醒** → 蓝牙/WiFi 自动恢复、屏幕亮起。

> Light Sleep 在倒计时和频谱运行时不会触发。

## 九、SD 卡 ROM 目录

Retro-Go 游戏系统从 SD 卡读取 ROM 文件，需要按以下结构建立目录：

```
/sdcard/roms/
├── nes/        → 存放 .nes 文件（红白机/FC）
├── gb/         → 存放 .gb .gbc 文件（Game Boy）
├── sms/        → 存放 .sms 文件（Sega Master System）
├── pce/        → 存放 .pce 文件（PC Engine）
└── snes/       → 存放 .smc 文件（Super Nintendo）
```

> SD 卡格式化为 FAT32。如果某个目录不存在，游戏系统不会显示该平台的选项。
> 建议先在电脑上建好目录再插入设备。

## 十、常见问题

| 问题 | 解决方法 |
|------|---------|
| 屏幕不亮 | 检查背光引脚 + SPI 连接；重新烧录 bootloader |
| 按键无反应 | 检查 GPIO 定义；看串口 `MAIN` 标签日志 |
| 天气不显示 | 设置 API Key了吗？WiFi 连上了吗？看 `WEATHER` 标签 |
| 蓝牙连不上 | 清除所有配对记录，重启蓝牙再试 |
| 电池一直 100% | Rev 2 PCB 无 ADC 检测，需自行加充电电路 |
| 设备反复重启 | 开机后等 5-10 秒再操作，等 WiFi/天气加载完 |
