# 操作手册 — ESP-Handheld 使用指南

## 一、按键说明

### 按键布局

```
              [START]                    [UP]
                                          ↑
    [LEFT] ←  [DOWN]  → [RIGHT]    [A]  [B]
                ↓
```

| 按键 | 物理位置 | GPIO | 功能 |
|------|---------|------|------|
| **UP（上）** | 左上 | GPIO16 | 上方向/上翻 |
| **DOWN（下）** | 左下 | GPIO15 | 下方向/下翻 |
| **LEFT（左）** | 中左 | GPIO17 | 左方向 |
| **RIGHT（右）** | 中右 | GPIO7 | 右方向/确认 |
| **A** | 右下 | GPIO4 | 确认/选择 |
| **B** | 右下（A左侧） | GPIO5 | 返回/取消 |
| **START** | 左上（UP上方） | GPIO6 | 菜单/开始 |

### 组合键

| 组合 | 按住时间 | 功能 |
|------|---------|------|
| **START + B（主界面）** | — | 退出当前应用/返回上一级 |
| **START + B（Retro-Go）** | 150ms | 从 Retro-Go 返回 ESP-Handheld |
| **START + B（XiaoZhi）** | 150ms | 从 XiaoZhi 返回 ESP-Handheld |
| **START + A（游戏中）** | — | 模拟 SELECT 键 |
| **START（按住）** | 1秒 | 打开游戏菜单（Retro-Go 中） |
| **RIGHT（键盘模式）** | — | 按住通话（PTT 语音） |

## 二、界面导航

### 主屏（Launcher）

开机后进入主屏，显示：
- **顶部状态栏：** WiFi 图标、当前时间、蓝牙状态、电池电量
- **中央天气卡片：** 天气预报（需要配置 API Key）
- **室内温湿度：** AHT20 传感器数据（如果硬件有此模块）

按 **START** → 进入应用菜单

### 应用菜单（Menu）

9 个应用卡片，2 列布局：
```
┌─────────┬─────────┐
│  游戏   │  小智   │
├─────────┼─────────┤
│  键盘   │  鼠标   │
├─────────┼─────────┤
│  占卜   │ 小六壬  │
├─────────┼─────────┤
│  计时   │  频谱   │
├─────────┼─────────┤
│  设置   │         │
└─────────┴─────────┘
```

- **UP/DOWN：** 上/下翻页
- **LEFT/RIGHT：** 左/右移动
- **A：** 选择并进入应用
- **B/START：** 返回主屏

### 应用内导航

- **B/START：** 退出当前应用，返回菜单（大部分应用）
- **A：** 确认操作

## 三、各应用使用说明

### 游戏（Retro-Go）

进入"游戏"卡片后，设备会**重启并切换到 Retro-Go** 游戏系统（ota_0 分区）。

**功能：**
- NES/FC 红白机模拟器（默认）
- GB/GBC 掌机模拟器
- SMS/GG 世嘉模拟器
- PCE/TurboGrafx-16 模拟器
- SNES/SFC 超任模拟器

**菜单导航：**
- 在 ROM 浏览器中通过 UP/DOWN 选择游戏
- 按 A 进入/确认
- 按 B 返回
- **START 按住 1 秒：** 游戏菜单（存档/读档/退出）

**返回 ESP-Handheld：** 在游戏菜单中选"Exit"，或 **START+B 按住 150ms**

> 注意：需要在 SD 卡的 `/sdcard/roms/` 下放置对应平台的 ROM 文件

### 小智（XiaoZhi AI）

进入"小智"卡片，设备会**重启并切换到 XiaoZhi AI**（ota_1 分区）。

XiaoZhi AI 是一个 AI 语音助手固件。如需返回，在 XiaoZhi 中按住 **START+B 150ms**。

### 键盘（BLE HID 键盘）

将设备作为**蓝牙键盘**连接到手机、平板或电脑。

**使用方法：**
1. 进入"键盘"应用
2. 在目标设备上打开蓝牙搜索"ESP-Handheld"
3. 点击配对（Just Works 配对，无需输入密码）
4. 配对成功后即可使用物理按键作为键盘输入

**按键映射：**
| 物理按键 | 键盘输出 |
|---------|---------|
| UP | 上方向键 ↑ |
| DOWN | 下方向键 ↓ |
| LEFT | 左方向键 ← |
| RIGHT | 右方向键 → |
| A | Enter（回车） |
| B | Esc（取消） |
| START | Tab |

**PTT 语音：**
按住 **RIGHT** 键可像对讲机一样说话，松开停止。语音通过 WiFi 实时传输到电脑（需先设置 IP 地址）。

**退出：** 同时按 **START + B**

### 鼠标（BLE HID 鼠标）

将设备作为**蓝牙鼠标**使用（通过 IMU 陀螺仪控制指针）。

**使用方法：**
1. 进入"鼠标"应用
2. 在目标设备上配对蓝牙
3. 握持设备上下左右倾斜来控制鼠标指针

**按键：**
- **A：** 鼠标左键
- **B：** 鼠标右键
- **UP：** 提高灵敏度
- **DOWN：** 降低灵敏度

**退出：** 同时按 **START + B**

### 占卜（易经）

基于《易经》六十四卦的占卜工具。

**使用方法：**
1. 进入"占卜"应用
2. 选择占卜方式（默认为六爻）
3. 按 A 开始摇卦
4. 系统随机生成卦象和变卦
5. 显示卦名、卦辞、爻辞解释

**退出：** 按 **B** 或 **START**

### 小六壬（Xiao Liu Ren）

传统小六壬占卜术的数字版本。

**使用方法：**
1. 进入"小六壬"应用
2. 输入三个数字（月、日、时）
3. 按 A 开始推算
4. 显示占卜结果（大安/留连/速喜/赤口/小吉/空亡）

**退出：** 按 **B** 或 **START**

### 计时（倒计时）

简单的倒计时工具。

**使用方法：**
1. 进入"计时"应用
2. 通过 UP/DOWN 设置时间
3. 按 A 开始/暂停
4. 倒计时结束后设备会提示

**退出：** 倒计时结束后按 A 重置，或同时按 **START + B** 退出

### 频谱（音频频谱分析仪）

实时 FFT 频谱可视化工具，显示麦克风捕捉到的音频频谱。

**功能：**
- 左右对称的频谱柱子
- 可调节参数：灵敏度、FFT 大小、平滑系数、颜色主题

**操作方法：**
- **UP/DOWN：** 选择参数项
- **LEFT/RIGHT** 或 **A：** 调整参数值
- 参数会自动保存

**退出：** 按 **B** 或 **START**

### 设置

系统设置菜单。使用 **UP/DOWN** 导航，**A** 进入子项，**B/START** 返回。

| 设置项 | 功能 |
|-------|------|
| 音量 | 调整扬声器音量（0-100） |
| 亮度 | 调整屏幕亮度（0-100） |
| 配置 WiFi | 进入 WiFi 配置模式（热点+网页配置） |
| 天气地区 | 选择和风天气的地区（泉州各县区） |
| 主题 | 切换主题配色（6 种内置） |
| 主题颜色 | 自定义 5 个颜色槽 |
| 按键映射 | 自定义 7 个按键的功能 |
| 待机时间 | 设置自动息屏/休眠时间（5秒-2小时） |
| 关于 | 显示系统信息 |

## 四、WiFi 配置

设备有两种方式连接 WiFi：

### 方法一：设置菜单配置
1. 进入 设置 → 配置 WiFi
2. 设备会关闭蓝牙，启动 WiFi 热点
3. 用手机/电脑连接热点 **ESP-Handheld_XXXX**
4. 浏览器打开 `192.168.4.1`
5. 选择你的 WiFi 并输入密码
6. 配置成功后设备自动连接

### 方法二：运行时配置
如果已经有保存的 WiFi 密码，设备会自动连接。

### WiFi 音频 PC IP 设置
1. 在主屏按 START → 进入设置 → IP 输入
2. 输入你电脑 IP 地址的最后一位
   - 例如：ESP 的 IP 是 192.168.1.100，电脑是 192.168.1.50
   - 只需输入 50
3. 设备会自动拼接完整的 IP 地址

## 五、蓝牙配对

### 首次配对（手机/平板/电脑）
1. 进入"键盘"或"鼠标"应用
2. 在目标设备上打开蓝牙设置
3. 搜索名为 **ESP-Handheld** 的设备
4. 点击配对（Just Works 模式，无需确认码）
5. 配对成功后，按键即可输入

### Windows 配对注意事项
- 部分 Windows 蓝牙适配器需要 MITM 配对
- 如果连接失败，先删除 Windows 中所有 "ESP-Handheld" 配对记录
- 重启蓝牙再试
- 查看串口日志中的 `BLE_HID` 标签，确认 `AUTH OK`

## 六、电源管理

### Light Sleep（轻睡眠）

设备在没有操作一段时间后（默认 5 分钟，可在设置中调整）：
1. 先关闭屏幕背光（息屏）
2. 自动关闭蓝牙（释放 BT 控制器）
3. 关闭 WiFi（关闭射频）
4. 进入 Light Sleep 模式

**唤醒：** 按任意按键立即唤醒
- 蓝牙自动重新初始化
- WiFi 自动重新连接
- 背光自动恢复

**估计功耗：** 正常工作 150-200mA → 待机 5-10mA

### Light Sleep 不会触发的场景
- 倒计时运行时
- 频谱分析界面运行时

## 八、固件切换

设备可运行三个独立的系统：

| 系统 | 分区 | 用途 | 切换方式 |
|------|------|------|---------|
| **ESP-Handheld** | factory | 主界面/天气/工具 | 开机默认，或从 Retro-Go/XiaoZhi 按 START+B |
| **Retro-Go** | ota_0 | 游戏模拟器 | 菜单 → 游戏 |
| **XiaoZhi AI** | ota_1 | AI 语音助手 | 菜单 → 小智 |

**返回 ESP-Handheld 的方法（从 Retro-Go 或 XiaoZhi）：**
同时按住 **START + B** 约 150ms，设备会重启回到主系统。

## 九、按键映射自定义

在 设置 → 按键映射 中，你可以重新分配 7 个物理按键的功能。

每个按键可以映射为：
- UP / DOWN / LEFT / RIGHT（方向键）
- A / B（动作键）
- START（开始）
- 以及其他功能

## 十、主题系统

### 切换主题
设置 → 主题 → 选择 6 种内置配色之一

### 自定义颜色
设置 → 主题颜色 → 分别调整 5 个颜色槽：
- **BG：** 背景色
- **ACCENT：** 强调色（按钮、高亮）
- **HIGHLIGHT：** 选中边框
- **TEXT：** 主要文字
- **SUBTEXT：** 次要文字

所有修改自动保存到设备 NVS。

## 十一、PC 端工具

### WiFi 音频接收器

设备可以将麦克风音频实时传输到电脑（需要电脑运行接收程序）。

**电脑端：**
```bash
python tools/run_voice_receiver.pyw
```

**设备端：**
1. 确保设备已连接 WiFi
2. 进入 设置 → IP 输入，输入电脑的 IP
3. 在键盘应用中按住 **RIGHT** 键说话
4. 电脑端会实时播放音频

### 发送 IP 地址

```bash
python tools/send_ip.py
```
通过串口向设备发送电脑的 IP 地址。

## 十二、常见问题

### 屏幕无显示/黑屏
- 检查背光引脚是否正确
- 检查 SPI 引脚连接
- 尝试重新烧录（包含 bootloader）

### 按键无反应
- 检查按键 GPIO 引脚定义是否匹配硬件
- 进入监控看 `MAIN` 标签是否有按键事件

### 天气不显示
- 是否已配置 API Key？（`idf.py menuconfig` 中设置）
- 是否已连接 WiFi？
- 检查串口 `WEATHER` 标签的日志

### 蓝牙配对失败
- 删除所有 "ESP-Handheld" 配对记录
- 重启蓝牙再试
- 查看 `BLE_HID` 标签日志

### 电池显示 100%
Rev 2 PCB 没有 ADC 检测电池电压。如需此功能，需要自行添加充电检测电路。

### 如何恢复出厂设置？
重新烧录 `ota_data_initial.bin` 到 0xd000 并重启。
